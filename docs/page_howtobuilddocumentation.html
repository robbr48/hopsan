How to Build the Documentation

The Hopsan Documentation is based on Doxygen. You are only required to install Doxygen, the other tools are optional.
Graphviz and dvipng (with latex) are used to show pretty dependence graphs and latex equations in the documentation.
If you want to build a pdf manual than LaTeX must be installed (and pdf activated in the Doxyfile)

Install and build the documentation in Ubuntu

  1. Install the necessary packages with: apt-get install doxygen graphviz dvipng
    To build or later rebuild the documentation, run "buildDocumentation.sh". Add the argument full to build full documentation.
    Open doc/html/index.html in a web browser after building.

Install and build the documentation in Windows

  1. (Required) The latest version of Doxygen can be downloaded here: http://www.stack.nl/~dimitri/doxygen/download.html#latestsrc
  2. (Optional) A LaTeX package including dvipng (to generate png for latex equations), We recommend: http://miktex.org/
  3. (Optional) You will also need Ghostscript 32-bit version 9.18 for formulas, install to default directory. C:/ProgramFiles/gs/gs9.18 or C:/ProgramFiles (x86)/gs/gs9.18
    You must use 9.18 because paths are hard-coded in the buildDocumentation.bat script, (you can add new path settings in the script if you want to use a newer version). You can find it here: http://ghostscript.com/download/gsdnld.html
  4. (Optional) The latest stable version of Graphviz (for cool graphs) can be downloaded here: http://www.graphviz.org/Download_windows.php
    You must use 2.38 because paths are hard-coded in the buildDocumentation.bat script, (you can add new path settings in the script if you want to use a newer version).

To build or later rebuild the documentation, double click the buildDocumentation.bat or buildFullDocumentation.bat file.
Open doc/html/index.html in a web browser after building.
